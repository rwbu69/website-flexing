Vanilla CSS

  • ✓ Kontrol penuh atas styling
  • ✓ Gak ada dependency eksternal
  • ✓ Ukuran file lebih kecil
  • ✓ Animasi custom sesuka hati
  • ✗ Butuh waktu lebih lama develop
  • ✗ Nulis CSS dari nol

Bootstrap

  • ✓ Development cepat
  • ✓ Komponen siap pakai
  • ✓ Grid system responsive
  • ✓ Browser compatibility bagus
  • ✗ Ukuran file lebih gede
  • ✗ Tampilan mirip-mirip kalo gak dimodif

Tailwind CSS

  • ✓ Utility-first, gampang banget
  • ✓ Highly customizable
  • ✓ Gak perlu pindah file CSS
  • ✓ Purge CSS otomatis (file kecil)
  • ✗ HTML jadi rame class
  • ✗ Perlu belajar utility classes
